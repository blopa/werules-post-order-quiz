## Installation

1. Clone or download this repository into your Magento 2 `app/code` directory:
   ```bash
   cd app/code
   git clone https://github.com/yourusername/Werules_PostOrderQuiz.git Werules/PostOrderQuiz
   ```

2. Enable the module:
   ```bash
   php bin/magento module:enable Werules_PostOrderQuiz
   php bin/magento setup:upgrade
   php bin/magento cache:flush
   ```

3. Deploy static content:
   ```bash
   php bin/magento setup:static-content:deploy -f
   ```
